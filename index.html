Defy: Audio Demake Tool for PICO-8

Drop audio file here or click to choose file.
Use this tool to download a raw, 8-bit, unsigned, 5512.5K PCM data file from any audio file.
To use the PCM file with PICO-8:
  1. In PICO-8, type load #defy at the command prompt.
  2. Then type run.
  3. Drag and drop the PCM file onto PIC0-8 to hear the clip.
  4. When the file is done playing, a compressed PCM string is copied to your computer's clipboard.
  5. The following code may be used in your own carts to play the PCM string.
